‚ö° Troubleshooting

Q: Nothing happens when I double-click start_server.bat?

A: PHP might not be installed. Install XAMPP from https://www.apachefriends.org or install PHP separately.

Q: Still getting network error?

A: Make sure the command window from start_server.bat is STILL OPEN. Close and reopen it.

Q: URL says file:// instead of http://?

A: That's the problem! You must use http://localhost:8000 not file://. Copy-paste the correct URL.

Q: Port 8000 already in use?

A: Edit start_server.bat, change 8000 to 8080 or 3000, save, and run again.
